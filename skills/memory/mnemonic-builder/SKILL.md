# Skill: Mnemonic Builder

Convierte información abstracta y difícil de recordar en estructuras mentales memorables y duraderas.

## Filosofía
El cerebro humano no evolucionó para recordar listas de datos abstractos, sino para recordar historias, lugares e imágenes impactantes. Esta skill aplica la neurociencia del aprendizaje para crear "ganchos" de memoria.

## Instrucciones para la IA
Cuando se active `/mnemonic-builder [lista/concepto/datos]`:

1. **Analiza el material.** ¿Son nombres, fechas, procesos o jerarquías?
2. **Propón 3 tipos de mnemotecnias:**
    - **Acrónimo/Acróstico:** Una palabra o frase fácil.
    - **Imagen Mental:** Una descripción visual vívida y extraña (cuanto más rara, mejor se recuerda).
    - **Historia/Narrativa:** Un micro-relato que encadene los elementos.
3. **Sugerencia de Palacio de Memoria.** Si el material es extenso, sugiere un recorrido por una casa o lugar conocido donde "colocar" los datos.
4. **Verificación de Recuperación.** Pide al usuario que intente repetir la mnemotecnia inmediatamente.

## Ejemplo de Prompt de Activación
"/mnemonic-builder los 12 pares craneales"
"/mnemonic-builder las etapas de la mitosis"
"/mnemonic-builder los principales filósofos presocráticos"
